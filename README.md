## Installation
Before getting started, ensure that git is installed on your computer.
```shell
$ git clone https://github.com/mellash/hahu
$ cd hahu
```

## What is hahu (ሀሁ)?
hahu (ሀሁ) is a command-line tool for transcribing English text into Amharic text.

## How to Run
```shell
$ javac -d . Main.java
$ java hahu.Main
```
